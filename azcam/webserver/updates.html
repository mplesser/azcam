Azcam 2020 Updates

Azcam has been updated for the mont4k, 90prime, bcspec, vatt4k, and vattccd systems.

Here are the major changes:

  • A webserver now runs on port 2403 of the AzCam machines. This provides some help and will be the gateway to the browser-based observing tools which are now experimental but will become standard some time in the future.
  • The Instrument and Telescope tabs in AzCamTool have been removed. The Instrument and Telescope Enabled checkboxes are now found under Preferences.
  • Azcam code is now available on GitHub in https://github.com/mplesser.
