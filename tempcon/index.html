    TempCon Class

    This class defines Azcam's temperature controller interface. The class is usually instantiated as the tempcon object in the server.

    class azcam.tempcon.TempCon(obj_id='tempcon', obj_name='Tempcon')

    Temperature control class.

    apply_corrections(self, temperature, temperature_id=0)

    Correct the temperatures for offset and scale is temperature correction is enabled.

    :param temperature: temperatures to be corrected :param temperature_id: temperature ID number

    define_keywords(self)

    Defines temperature control keywords if they are not already defined.

    delete_keyword(self, keyword)

    Delete a keyword.

    exposure_finish(self)

    Custom commands after exposure finishes.

    exposure_start(self)

    Custom commands before exposure starts.

    get_all_keywords(self)

    Return a list of all keyword names.

    get_control_temperature(self, temperature_id=0)

    Get the control temperature (set point).

    :param temperature_id: temperature ID number

    get_info(self)

    Returns header info. Returns [Header[]]: Each element Header[i] contains the sublist (keyword, value, comment, and type). Example: Header[2][1] is the value of keyword 2 and Header[2][3] is its type.

    get_keyword(self, keyword)

    Read a temperature keyword value and returns it as [value, comment, type string].

    :param keyword: keyword name to read

    get_temperature(self, temperature_id=0)

    Returns a system temperature.

    :param temperature_id: temperature ID number

    get_temperatures(self)

    Return all system temperatures.

    initialize(self)

    Initialize the temperature control interface.

    read_header(self)

    Returns the current header. Returns [Header[]]: Each element Header[i] contains the sublist (keyword, value, comment, and type). Example: Header[2][1] is the value of keyword 2 and Header[2][3] is its type.

    reset(self)

    Reset tempcon object.

    set_calibrations(self, cals)

    Set calibration curves for temperature sensors.

    :param cals: list of flags defining each temperature sensor type

    The values of these flags are from the list below which define the calibration curves to use for each sensor's temperature conversion.

    • 0 => DT670 diode
    • 1 => AD590 sensor
    • 2 => 1N4148 diode
    • 3 => 1N914 diode
    set_control_temperature(self, temperature=None, temperature_id=0)

    Set the control temperature (set point).

    :param temperature: control temperature in Celsius. If not specified, use saved value :param temperature_id: temperature ID number

    set_corrections(self, temperature_offsets=None, temperature_scales=None)

    Set temperature correction values and enable correction. If both parameters are None then correction is disabled.

    :param temperature_offsets: list of offsets for each temperature :param temperature_scales: list of scale factors for each temperature

    set_keyword(self, keyword, value, comment=None, typestring=None)

    Set a keyword value and comment. typestring is one of 'str', 'int', or 'float'.

    update_header(self)

    Update the header, reading current data. Deletes all keywords if the object is not enabled.
