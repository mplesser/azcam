    Controller Class

    This class defines Azcam's camera controller interfaces.

    azcam.controller.Controller

    The base controller tool. Usually implemented as the "controller" tool.

    Attributes:

    Name Type Description
    self.interface_type int

    0 = demo, 4 = PCIe
    self.header Header object

    controller header
    self.exposure_time float

    exposure time (secs)
    self.detpars DetPars object
    self.reset_flag bool

    True if the controller has been reset

    __init__(self, tool_id='controller', description=None) special

    Parameters:

    Name Type Description Default
    tool_id

    tool name

    		 'controller'
    description

    description of this tool

    		 None

    flush(self, cycles=1)

    Flush or clear out the sensor. Returns after clearing is finished, which could take many seconds.

    Parameters:

    Name Type Description Default
    cycles

    number of times to flush sensor

    		 1

    set_roi(self)

    Sets ROI parameters values in the controller based on focalplane parameters.

    set_shutter(self, state, shutter_id=0)

    Open or close controller shutter.

    Parameters:

    Name Type Description Default
    state bool

    True to open shutter, False to close shutter

    		 required
    shutter_id int

    shutter ID number

    		 0

    set_shutter_state(self, flag=0)

    Sets the shutter state during an exposure.

    Parameters:

    Name Type Description Default
    flag bool

    True open shutter during exposure, False close shutter during exposure

    		 0
