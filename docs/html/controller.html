Controller Class¶

This class defines Azcam’s camera controller interfaces.

class Controller(obj_id='controller')¶

Bases: object

Base controller class for Azcam.

delete_keyword(keyword)¶

Delete a keyword.

enabled¶

True when controller is enabled

get_keyword(keyword)¶

Return a keyword value and its comment. Comment always returned in double quotes, even if empty.

initialized¶

True when contorller is initialized

interface_type¶

interface type (0 = demo, 4 = PCIe)

name¶

controller name

obj_id¶

controller ID

set_keyword(keyword, value, comment=None, typestring=None)¶

Set a keyword value and comment. typestring is one of ‘str’, ‘int’, or ‘float’.

set_roi()¶

Sets ROI parameters values in the controller based on focalplane parameters.

set_shutter(state, shutter_id=0)¶

Open or close controller shutter.

set_shutter_state(Flag='close')¶

Sets the shutter state during an exposure. Flag is: open -> close shutter during exposure close -> open shutter during exposure

stop_idle()¶

Stop idle clocking.

update_header()¶

Update header. Normally controller keywords are set during reset.
