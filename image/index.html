    Image Class

    This class defines Azcam's image object. Within azcam it is also used to define the exposure.image object which receives image data from a camera controller.

    The Image class is imported automatically into the azcam namespace from the azcam.image module.

    Usage Example:

    im1 = azcam.Image('test.fits')

    azcam.image.Image

    Class to create and manipulate the standard azcam image object.

    Type Description
    remote_imageserver_flag

    read_file(self, filename)

    Read image from file. This version only supports FITS/MEF.

    send_image(self, local_filename)

    Send local_filename image to a remote image server. This method may run in an async thread. Server types (self.server_type) are: azcam, lbtguider, and dataserver.

    set_remote_imageserver(self, remote_imageserver_host='', remote_imageserver_port=0, remote_imageserver_filename='image')

    Set parameters so image files are sent to a remote image server. If no host is provided then reset flag to local image file.

    set_scaling(self, gains=None, offsets=None)

    Set gains and offsets for image assembly. Use None for no scaling.

    write_file(self, filename, filetype=-1)

    Write image to disk file. filetype is 0 for FITS, 1 for MEF, 2 for BIN, 6 for assembled.
